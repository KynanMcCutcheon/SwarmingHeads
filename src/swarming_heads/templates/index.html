Welcome!

If you are seeing this, it means you've successfully started up Django!

This url will eventually be the home page to the Swarming Heads web interface. For the moment, I have built a couple of examples of some basic django things.

As we learn to do things in Django, I think this should be up a nice collection of 'How-tos' and might even be able to serve as part of our user documentation. (In case others want to modify the web interface after the project is over)

Currently, the examples available include:

  • Example of getting parts of the url
  • Demo of djangos templating system
  • Handling forms in Django
  • Using jQuery Ajax with Django
  • Admin page
  • Admin documentation page

At the moment, all those test pages (except the admin pages) are being held in an app named 'testing'.

Currently, there are some key directories in the project:

  • apps - This will hold all our project apps
  • db - This is where any sqlite3 databases can be stored
  • media - This is where we can put all our static resources (images, javascript, css etc)
  • static - This is for when we move onto our clients server. It will hold all static resources. Can be ignored for now
  • templates - Our master page templates for making reusing static elements where possible
